# Changelog

All notable changes to the "dbooly" extension will be documented in this file.

## [0.0.1] - 2024

### Added
- Initial release
- MySQL/MariaDB connection management
- Secure credential storage
- SQL execution with Ctrl+Enter
- CodeLens for running individual statements
- Schema browser with table/column exploration
- Table data viewer
- SQL auto-completion via Language Server
- SQL formatting with dialect awareness
- Real-time SQL diagnostics
- Query cancellation support
- Script management sidebar
